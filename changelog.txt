Version: 1.1.0
Date: 2026-07-14
  Features:
    - Initial release for Factorio 2.1.
    - Deconstruction marker: an item with no crafting recipe, placed in blueprints on tiles whose entities should be removed.
    - Super-force building a blueprint with markers marks the underlying entities for deconstruction; the marker ghosts remove themselves and never request materials.
    - Shortcut bar button and Alt+X keybind put a marker in the cursor; the cursor refills after each placement.
